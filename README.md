# CC Game Engine Specifications

A Rust-based game engine designed for couch co-op tactical strategy games, with a focus on hex-based combat systems similar to tabletop D&D.

## Project Goals

- **Ease of Use**: Quick prototyping of new tactical games
- **Couch Co-op First**: Local multiplayer as the primary use case
- **Tactical Strategy**: Hex-based combat with D&D-style mechanics
- **Hot Reloading**: Rapid iteration without recompilation

## Documentation Structure

- `/specs/architecture/` - High-level system design
- `/specs/features/` - Feature specifications
- `/specs/api/` - API and interface designs
- `/examples/` - Example game scenarios and use cases

## Target Features

See individual spec documents for details on:
- Hex grid system
- Tactical combat mechanics
- Character/unit systems
- Environmental interactions
- AI and pathfinding
- Rendering and UI
